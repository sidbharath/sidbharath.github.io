3. Automatic Retrieval of Results from NTA

In recent days, the NTA website has frequently changed the captcha when we attempt to download students' results. This makes automation using VBA coding or macros quite challenging. We would like to understand whether this process can be automated using any advanced AI technology or coding.

Simplest Solution
Level 2: Intelligence
Captcha-Solving Options:
  • Option 1 - Captcha Solving Service:
    • Use 2Captcha or Anti-Captcha API
    • Python script sends captcha image → Service solves → Script continues
    • Cost: ~$1-3 per 1000 captchas
  • Option 2 - Browser Automation with Human:
    • Use Playwright/Puppeteer to automate everything EXCEPT captcha
    • Script pauses, human solves captcha, script continues
    • Still saves 90% of time
  • Option 3 - AI Captcha Solver:
    • If captcha is text-based: Use GPT-5 Vision to read it
    • Success rate varies (60-90% depending on captcha type)
